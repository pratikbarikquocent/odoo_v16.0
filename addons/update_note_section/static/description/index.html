Update Section and Note in LOCKED Status of Sale Order and Purchase Order for specific Users.

User can update Section and Note in POSTED Status of Invoice.

For more details please see the video and check below screenshots

Giving Access To Specific User

Assign access(Update Section & Note in Locked Sale Order) to the User for updating sale order's section value. UPDATE SECTION Button visible only for the Users who assigned the access rights.

'Update Notes and Section' Action menu.

Click on Update Notes and Section Action it will open form view with existing section and note values.

User can update section and note values by clicking on Update Button, value will be change in lines.

Section and note are now updated.
