Odoo 16 Receipt Order Linkage with Purchase Order

This Odoo module customizes the Inventory module by linking it with Purchase Orders and adding a Purchase Order action button in the Receipt Order view.

KEY HIGHLIGHTS

Purchase Button in Inventory

Add a button in the Receipt Order form view (Inventory module) that redirects to the associated Purchase Order!

Purchase Count In Receipt Order

Automatically calculates and shows the count of related Purchase Order on Receipt Orders!

Find In Search View

Easily we can find Receipt Orders by searching for the Purchase order reference in the search view!

Connect The Module Effortlessly

Enter into Receipt Order

Open the Inventory module from the dashboard -> Select the receipt order from the Inventory overview.

Find Purchase source destination in a snap

We can search the required Purchase Order in search view by searching source destination!

Easily find the desired Purchase Order in the search view -> Simply enter the order reference to search.

Instantly tally up your related Purchase on Receipt Orders

Automatically calculates and shows the count of related Purchase on Receipt Orders!

Navigate to the Purchase Order -> There is now a button that allows you to directly view the Purchase Order.

Installation Instructions

  1. Download the Module
    • Download the Action Button In RO Navigate To Linked PO.
    • Save the downloaded module file (usually a .zip file) to your Odoo addons.
    • Log in to your Odoo system as an Administrator.
    • Navigate to the Apps module on your Odoo dashboard.
  2. Install the Module
    • Once the module is uploaded, it will appear in the Apps list.
    • Click the Install button next to the Action Button In RO Navigate To Linked PO.
    • Odoo will install the module for you. Once it's done, the module is ready to use!
